Hot Keys		 X Northern Captain provide extension file association and execution thought simple file of extensions. This is text file each line of that describe file extension begining from dot '.' and application assocated with this extension. XNC reads this file at startup and then looks for file with following extensions in directory, highlights them with 'extension color', and allows you execute given application with file by pressing on it with 'execution key' [Enter]. If more than one application defined for this extension (two or more lines in xnc.ext) choice menu appears and You can choose needed command. 

Format of the extension string:

#Put comments after this sign
.extension: application %s #comment if need
.extension: application %s/%s #comment if need

where:

  %s - current file name without path. Application run in current directory with current file. 
  %s/%s - current file with full path. Application run in opposite panel directory with full path file. 

To add/edit extensions file choose in upper menu 'Option' then 'Edit extension file'. Remember that if you are using external editor (not IVES) then you need to restart xnc for your changes will take effect.
